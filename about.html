About

I’m Daniel, a web designer and developer and the founder of Carson Web Studio. I design and build modern, responsive websites with a focus on clarity, usability and thoughtful visual design.

Before moving into web development, I spent several years working as a natural light portrait and fashion photographer. That experience developed my eye for composition, colour and visual storytelling — skills that now influence the way I approach web design.

I recently completed a Full Stack Software Development Diploma with Code Institute, where I developed practical skills in building responsive websites and web applications using HTML, CSS and JavaScript.

Through Carson Web Studio, I enjoy helping churches, charities and community organisations create websites that clearly communicate who they are and make it easy for people to connect with them.

My Approach

  • Clarity: Making sure visitors can easily find the information they need.
  • Visual Design: Creating layouts that are clean, balanced and engaging.
  • Usability: Building responsive websites that work well across all devices.
